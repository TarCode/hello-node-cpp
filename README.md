# Hello World NodeJS C++ Addon Example

Example of NodeJS addon written C++.

Based on https://nodejs.org/dist/latest-v8.x/docs/api/addons.html

```
git clone https://github.com/TarCode/hello-node-cpp
```

```
cd hello-node-cpp
```

### To configure
```
node-gyp configure
```

### To Build
```
node-gyp build
```

### To run
```
node main.js
```
